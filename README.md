# gasless-intents-examples

Example repository with scripts for creating gasless intents.

## Running the repo

Install dependencies by runnig:

```
nvm use
npm install
```

## Running the scripts

You can run the individual scripts using `npx tsx`:

```
npx tsx src/gasless-intents/submit-bundle.ts
npx tsx src/gasless-intents/get-bundle-by-id.ts
npx tsx src/gasless-intents/list-orders.ts
```
